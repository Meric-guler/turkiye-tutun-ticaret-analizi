# Turkey Tobacco Products Foreign Trade Analysis (2021–2025)

## Project Overview
This project analyzes Turkey’s import and export data for tobacco and tobacco-related products between 2021 and 2025. The goal is to understand trade volumes, product distribution, and year-over-year changes, and present the findings in a clear, easy-to-read format.

## Data Source
- *UN Comtrade Database*
- Coverage: HS codes 2401, 2402, 2403
- Time period: 2021 to 2025

## Tools & Workflow
1. *Data Preparation*: Cleaned and structured raw data using Google Sheets
2. *Analysis*: Used Python and Pandas to check data quality, fix formatting issues, and calculate total trade values and quantities
3. *Visualization*: Built an interactive report in Power BI to show trends, product breakdown, and trade balance
4. *Documentation*: Organized all files and results for easy review

## Main Findings
- Overall trade volume increased steadily over the 5-year period
- Raw tobacco and finished cigarettes make up the largest share of both imports and exports
- Export volumes grew faster than imports in recent years
- The data shows clear seasonal and yearly patterns in the sector
