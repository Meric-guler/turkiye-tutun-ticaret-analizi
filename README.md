# Turkey Tobacco Products Foreign Trade Analysis (2021–2025)

## Project Overview
This project examines Turkey’s foreign trade in tobacco and tobacco products from 2021 to 2025. The main aim is to analyze import and export volumes, see how trade is distributed across product groups, and track changes over the years. All calculations were cross-checked using different tools to make sure the results are consistent and reliable.

## Data Source
- UN Comtrade Database
- Coverage: HS codes 2401, 2402, 2403
- Time period: 2021 – 2025

## Tools & Workflow
- *Data Preparation*: Cleaned, formatted, and organized raw data using Google Sheets
- *Analysis*: First processed and validated the data with Python and Pandas. Then wrote SQL queries to run the same aggregations and calculations again — this helped confirm that the numbers were correct
- *Visualization*: Built an interactive report in Power BI to display trends, product breakdown, and trade balance in a clear way
- *Documentation*: Structured all files, code, and outputs so they are easy to follow and review

## Main Findings
- Total trade volume grew steadily over the five-year period
- Raw tobacco and finished cigarettes make up the largest share of both imports and exports
- Exports increased at a faster rate than imports in the last two years
- Results from Python and SQL matched closely, confirming the accuracy of the analysis
